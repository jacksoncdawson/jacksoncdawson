## Hi, I'm Jack 👋

Data engineer wrapping up an M.S. in Data Science & AI at USF. I build end-to-end — cloud data pipelines, ML systems, and the occasional model trained from scratch — and tend toward work that's genuinely useful to myself or my people.

### Projects

**[trndly](https://github.com/jacksoncdawson/trndly) — cloud data pipeline & platform**
A monthly batch ML pipeline that forecasts fashion trends for secondhand resellers: 4 retailer catalogs → versioned Parquet → two forecasting models, served to a React app. Full GCP stack as Terraform IaC, a private MLflow on Cloud Run, and 275+ tests gated in CI.

**[Grocery God](https://github.com/jacksoncdawson/Grocery-God) — serverless ETL pipeline**
Scrapes weekly grocery ads with Playwright, cleans them in Pandas, and lands them in Supabase. Runs as a Dockerized AWS Lambda on a schedule (EventBridge → ECR → S3), with a Streamlit logger up front.

**[chord5](https://github.com/jacksoncdawson/chord5) — music generation from scratch**  *(in progress)*
A ~5M-parameter decoder-only transformer that generates deadmau5-style 8-bar MIDI loops — custom REMI-lite tokenizer, a three-stage training curriculum, and a hand-curated fine-tune set. Currently narrowing the task to chord progressions.
